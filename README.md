# open-science-skills

This repository contains a collection of agentic skills for scientific research. These skills can be installed via the `skills` npm module, or just by copying them to your agents skills specific directory. 


## Installation via `skills`

```bash
npx skills add directory_containing_skills

# example
npx skills add ./git-rules
```
